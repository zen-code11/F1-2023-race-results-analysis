# Formula 1 2023 Race Analysis

## Overview
Exploratory Data Analysis of the 2023 Formula 1 race results dataset to analyze driver performance, team standings, and race characteristics.

## Key Analysis
- Constructor standings
- Driver standings
- Driver consistency
- Track difficulty (DNF analysis)
- Positions gained (overtaking performance)
- Correlation analysis

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
